Easy to Use

// Listen for events
PUBNUB.subscribe({
  channel: "my_channel",
  message: function (message) {
    console.log(message);
  }
});

// Send Events
PUBNUB.publish({
  channel: "my_channel",
  message: "Hello World!"
});
And now how is this built on our system? It is simple: pub and sub. We do the rest!
